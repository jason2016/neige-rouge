# Neige Rouge（红雪）在线点餐网站

## 项目概况

为巴黎中餐馆 **Neige Rouge（红雪）** 打造的在线菜单展示与点餐网站，支持中法双语。

## 客户信息

- **公司名称**: XUEHONG SARL
- **地址**: 7 rue des Ursulines, 75005 Paris

## 公网地址

<https://jason2016.github.io/neige-rouge/>

## 技术栈

- **前端框架**: Vite 5 + React
- **部署平台**: GitHub Pages

## 当前版本

**v1.0.0** — 静态菜单展示 + 点餐界面，中法双语

## 路线图

| 版本 | 功能 |
